Public data plane is not these URLs — browsers hit Caddy:443 → Janus router/hub. The /1.0 API lives on the control unix socket (Incus-style). Caddyfile decides admission into Janus; JSON decides routing inside Janus.

Internal engines

  • Registry — app/host/upstream maps; first-wins hosts
  • Router / proxy — host lookup → lb → reverse_proxy; else 404
  • Health — heartbeat clock + optional probes; unhealthy → 502/503
  • Allowlist — On-Demand/ACME ask callback; CertMagic obtains certs
  • Hub — WS hold, membership, bridge POST, publish fan-out
  • mDNS — ads from expose + hosts

v1 cut line

Ship first: GET /1.0, apps CRUD, PUT upstreams, heartbeat, DELETE. Defer: drain, events stream, separate host allow (fold into POST/PUT apps), hub diagnostics.
